# Сценарии запуска окружения IMIXS Microservice
* `source ./env` добавить сценарии в PATH
* `compose-command` подготовить сценарии запуска (перезапишет уже созданные, осторожно!)
* `imixs-up` запустить сервис документооборота
* `imixs-down [-p]` выключить сервис документооборота (-p - уничтожить данные выполнения)
